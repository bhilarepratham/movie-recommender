# üé¨ CineMatch - Movie & TV Recommender

A powerful, AI-driven movie and TV series recommendation engine built with Streamlit. CineMatch helps you discover your next favorite watch using mood-based filtering, advanced search, and real-time trending data.

## ‚ú® Features

- **Mood-Based Recommendations**: Find movies based on how you feel (Happy, Sad, Thrilled, etc.).
- **Smart Search**: Search for Movies, TV Series, and entire Collections (Franchises).
- **Rich Metadata**: View high-quality posters, backdrops, full plot summaries, cast & crew, and ratings.
- **Streaming Availability**: See where to watch titles (Netflix, Prime Video, Hulu, etc.) in real-time.
- **Dark Theme**: A premium, sleek dark user interface for the best viewing experience.
- **Analytics Dashboard**: Visualize data about movie ratings, genres, and languages.
- **Multilingual Support**: Explore content in various languages, including a dedicated section for Indian Cinema.

- **Frontend**: Streamlit

  - **OMDB (Open Movie Database)**: For additional metadata.
